Управление через админского бота

Все функции управления доступны через отдельного админского бота в Telegram

Статистика и мониторинг
/stats - Общая статистика системы
/users - Последние пользователи
/prizes - Ожидающие призы
/automation - Статистика автоматизации
/wins_stats - Канал выигрышей
Управление системой
/stars 123456789 +500 - Добавить звезды
/set_prize 123456789 cert "Сертификат"
/add_channel @channel "Название" 50 24
/wheel_settings mega - Настройки рулетки
/broadcast Сообщение - Рассылка
Как получить доступ к админскому боту:
  1. Настройте переменную окружения ADMIN_BOT_TOKEN
  2. Добавьте ваш Telegram ID в ADMIN_IDS
  3. Найдите бота по токену и отправьте команду /start
  4. Используйте команду /help для полного списка команд
